# Information

This is a simple console app that works as a to-do list. It forms part of my node.js learning process

```
==========================
|    Select an option    |
==========================
? What do you want to do? (Use arrow keys)
> 1. Create a task
  2. List tasks
  3. List completed tasks
  4. List pending tasks
  5. Complete task(s)
  6. Delete task(s)
  0. Exit
```
